# SSD - v2

## SSD, c'est quoi ? 

SSD est un outil de création de seedbox, tout en docker, et permet de déployer facilement tous les outils nécessaires à sa gestion :

Liste non exhaustive : 

- plex
- rutorrent
- *arr (radarr, sonarr, lidarr, bazarr, ...)
